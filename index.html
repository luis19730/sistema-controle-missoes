                    Passo a passo:

                    1. Acesse console.firebase.google.com

                    2. Clique em "Criar projeto" (pode usar o nome que quiser)

                    3. No menu lateral, clique em "Realtime Database" → "Criar banco de dados"

                    4. Escolha uma região e clique em "Ativar"

                    5. Na aba "Regras", cole:

                    {
  "rules": {
    ".read": true,
    ".write": true
  }
}
